--------------------------------------------
Структура dat-файлов для прямоугольной сетки
--------------------------------------------

inf2tr.dat (текстовый файл):
  kuzlov - число узлов сетки
  ktr    - число конечных элементов (прямоугольников)
  kt1    - число узлов с первыми краевыми условиями
  (остальные параметры для выполнения лабораторной работы не используются)


mu (текстовый файл)
  структура i-й строки:
	номер_материала  соответствующее_значение_мю
		(генерируются в соответствии с файлом sreda)


toku (текстовый файл)
  структура i-й строки:
  	номер_материала  соответствующее_значение_тока
		(генерируются в соответствии с файлом sreda)


nvkat2d.dat (двоичный файл):
  структура i-й записи (i=1..ktr):
  	1*long (m),
		где m - номер материала i-го прямоугольника в соответствии с файлами sreda и mu


nvtr.dat (двоичный файл):
  структура i-й записи (i=1..ktr): 
	6*long (i1,i2,i3,i4,0,1),
        	где i1,i2,i3,i4 - глобальные номера вершин i-го прямоугольника
		(левый верхний, правый верхний, левый нижний, правый нижний)


rz.dat (двоичный файл):
  структура i-й записи (i=1..kuzlov): 
	2*double (x,y),
		где x,y - (x,y)-координаты i-й вершины


l1.dat (двоичный файл):
  структура i-й записи (i=1..kt1):
	 1*long (k)
		где k - глобальный номер i-й вершины с первым нулевыми краевым условием




----------------------------------------------------------------------------
Структура dat-файлов для треугольной сетки
(для выполнения лабораторной работы сетка и соответствующая структура данных 
формируются из структура данных для прямоугольной сетки)
----------------------------------------------------------------------------

inf2tr.dat (текстовый файл):
  kuzlov - число узлов сетки
  ktr    - число конечных элементов (треугольников)
  kt1    - число узлов с первыми краевыми условиями
  (остальные параметры для выполнения лабораторной работы не используются)

mu (текстовый файл)
  структура i-й строки:
	номер_материала  соответствующее_значение_мю
		(генерируются в соответствии с файлом sreda)


toku (текстовый файл)
  структура i-й строки:
  	номер_материала  соответствующее_значение_тока
		(генерируются в соответствии с файлом sreda)


nvkat2d.dat (двоичный файл):
  структура i-й записи (i=1..ktr):
  	1*long (m),
		где m - номер материала i-го треугольника в соответствии с файлами sreda и mu


nvtr.dat (двоичный файл):
  структура i-й записи (i=1..ktr): 
	4*long (i1,i2,i3,nmat),
        	где i1,i2,i3 - глобальные номера вершин i-го треугольника,
		    nmat     - номер материала i-го треугольника (формируется из файла nvkat2d.dat)


xy.dat (двоичный файл, копируется из файла rz.dat):
  структура i-й записи (i=1..kuzlov): 
	2*double (x,y),
		где x,y - (x,y)-координаты i-й вершины


l1.dat (двоичный файл):
  структура i-й записи (i=1..kt1):
	 1*long (k)
		где k - глобальный номер i-й вершины с первым нулевыми краевым условием
